# SELIX: Um Modelo Matemático para a Taxa Selic Ideal

**Autores:** Zeh Sobrinho, GOS3

**Data:** 01/05/2026

## Resumo

Este artigo apresenta a SELIX, um modelo matemático que calcula a Taxa Selic ideal com base em 5 teoremas formalmente provados.

## Metodologia

### Z3 SMT Solver
Utilizamos o provador de teoremas da Microsoft Research para validar a consistência do sistema.

### Lean 4
Provador interativo que constrói uma prova passo a passo.

### Dados
- IPCA-12: 4.48% (BCB SGS 13522)
- ROE IBOVESPA: 31.23% (Ranking B3)

## Resultados

**SELIX = 9.25%** com juro real de 4.77%, garantindo Investment Grade.

## Conclusão

A Selic deveria ser reduzida de 14.50% para 9.25% em 10.5 meses.
